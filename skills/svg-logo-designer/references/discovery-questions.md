# Discovery Questions Reference

Complete question bank for logo design interviews. Select questions based on context -- never ask all of them. Organize into 2-3 focused rounds.

## Tier 1: Essential (Always Ask)

Non-negotiable. Ask all of these in Round 1:

1. **What is the exact brand name?** Include capitalization, spacing, registered marks (TM, R). Is there a tagline that should appear with the logo?
2. **What does the business do?** One sentence. Industry, product or service, what makes it different.
3. **What type of logo do you envision?** Present options:
   - Wordmark (text-only: Google, FedEx)
   - Lettermark (initials: IBM, NASA)
   - Icon/symbol (Apple, Nike swoosh)
   - Icon + text combination (Spotify, Slack)
   - Emblem (text inside a shape: Starbucks)
   - Abstract mark (geometric: Pepsi, Airbnb)
   - Not sure / open to suggestions
4. **Pick 3-5 words that describe how the brand should feel.** Prompt with examples if needed: bold, elegant, playful, trustworthy, innovative, warm, sharp, organic, futuristic, heritage, clean, powerful, friendly, sophisticated, rebellious, calm, dynamic, premium, approachable, precise.
5. **Do you have existing brand colors?** If yes, get hex codes. If no: "Any color direction or strong preferences? Any colors to absolutely avoid?"

---

## Tier 2: Important (Ask in Round 2)

Select the most relevant based on Tier 1 answers:

6. **Who is your target audience?** Age range, demographic, professional vs consumer, geographic region, psychographic profile.
7. **Name 2-3 logos you admire.** From any industry. Ask: "What specifically do you like about each? Is it the color, the shape, the feeling, the typography?"
8. **Who are your main competitors?** Helps ensure differentiation. "Is there a competitor's logo you think is particularly effective? Any that feel generic or off-brand?"
9. **What visual style resonates?** Present options:
   - Minimalist / clean
   - Geometric / mathematical
   - Organic / flowing
   - Bold / heavy-weight
   - Elegant / thin / refined
   - Playful / quirky
   - Tech / digital / futuristic
   - Vintage / retro / heritage
   - Brutalist / raw
10. **Any specific symbols, objects, or metaphors the logo should evoke?** Examples: "We're in cloud computing, so something sky/cloud related" or "We want to suggest speed and precision." Also: "Anything the logo should definitely NOT include?"
11. **Where will this logo primarily appear?** Website, app, business cards, packaging, signage, merchandise, social media, all of the above?

## Tier 3: Refinement (Ask in Round 2 or 3)

12. **What backgrounds will the logo sit on?** White, dark, colored, photographic, varied?
13. **Does it need to work at very small sizes?** Favicon (16px), app icon (64px), or mostly large-format?
14. **How many distinct concepts do you want to see?** Recommend 3-5. More = broader exploration, fewer = deeper refinement per direction.
15. **Which layout formats do you need?**
    - Horizontal (website header, email signature)
    - Vertical (social media, app icon)
    - Square (favicon, avatar)
    - Icon-only (watermark, small spaces)
    - Text-only (editorial)
16. **Is this a new brand or a rebrand?** If rebrand: "What should change? What must be preserved? Is there equity in the current mark?"

## Tier 4: Situational (Logo-Type Specific)

### For Wordmarks / Lettermarks
17. **Font style preference?** Serif (classic, editorial), sans-serif (modern, clean), slab-serif (strong, grounded), display (unique, expressive), script (personal, artisan)?
18. **All caps, title case, or lowercase?** Each conveys differently: CAPS = authority, Title = approachable, lowercase = modern/casual.
19. **Any custom letter modifications?** Ligatures, stylized characters, embedded symbols within letters?
20. **How important is readability vs uniqueness?** "Should someone be able to read it instantly, or is it OK if the design takes a second to parse?"

### For Icons / Symbols / Abstract Marks
21. **Literal or abstract?** Should viewers immediately recognize the object, or is a more interpretive/geometric approach preferred?
22. **Complexity tolerance?** Simple silhouette (Nike) vs moderate detail (Apple) vs intricate (Starbucks siren)?
23. **Should the icon tell a story or just signal the brand?** Narrative marks (FedEx hidden arrow) vs pure identity marks (Nike swoosh).
24. **Single symbol or composed of multiple elements?** Monolithic form vs layered/connected elements.

### For Combination Marks
25. **Icon-text relationship?** Icon left of text, above text, integrated/overlapping, or flexible?
26. **Should the icon work independently?** If yes, it must stand alone at small sizes without text.
27. **Hierarchy preference?** Icon dominant, text dominant, or balanced?

### For Emblems
28. **Shape preference?** Shield, circle, badge, ribbon, crest, hexagon, or something custom?
29. **How much detail is acceptable?** Emblems tolerate more detail but need simplified versions for small sizes.
30. **Classic or modern emblem?** Traditional crest with ornament vs clean modern badge.

## Tier 5: Industry-Specific Deep Dives

Ask only when the industry calls for specialized understanding:

### Tech / SaaS
31. "Should it feel enterprise-grade or consumer-friendly? Developer-facing or end-user-facing?"
32. "Does it need to sit well inside a product UI (app icon, sidebar) or mainly external (marketing, website)?"
33. "Any technical concepts that should be visually suggested? (connectivity, data, automation, cloud, AI)"

### Food / Restaurant / Beverage
34. "Should it evoke appetite? Artisan quality? Fast and convenient? Farm-to-table?"
35. "Will it appear on packaging, menus, or signage at large scale?"
36. "Does it need to feel premium or accessible?"

### Healthcare / Wellness
37. "Should it feel clinical and precise, or warm and nurturing?"
38. "Are there regulatory considerations for healthcare branding in your market?"
39. "Digital health / telemedicine vs traditional practice -- which end of the spectrum?"

### Luxury / Fashion
40. "Should it feel exclusive and aspirational, or accessible luxury?"
41. "Will it be stamped, embossed, or foiled? (affects complexity tolerance)"
42. "Heritage brand or contemporary launch?"

### Sustainability / Green
43. "How prominently should eco-messaging feature? Subtle or overt?"
44. "Should it look 'green' (earth tones, leaves) or signal sustainability through modern minimalism?"

### Creative / Agency
45. "Should the logo itself demonstrate creative capability, or be a clean vessel for the work?"
46. "Is there a specific creative discipline that should be signaled? (design, film, music, advertising)"

### Finance / Legal
47. "How much should tradition vs innovation be signaled?"
48. "Is this B2B (institutional trust) or B2C (approachable finance)?"

### Education / EdTech
49. "Academic rigor or modern learning? Traditional institution or startup?"
50. "Young learners, adult education, or professional development?"

### Sports / Fitness
51. "Dynamic and aggressive, or disciplined and focused?"
52. "Team/community identity or individual brand?"

## Tier 6: Emotional & Psychological Probing

Use these to extract deeper brand DNA when Tier 1-2 answers feel surface-level. These questions bypass the analytical mind and access gut-level brand intuition.

### Sensory Association
53. **"If your brand were a texture, what would it feel like?"** (Smooth glass? Rough canvas? Warm wood? Cold steel? Soft silk?)
54. **"If your brand were a sound, what would it be?"** (A deep bass note? A crisp click? A gentle chime? A roaring engine?)
55. **"If your brand walked into a room, what would people notice first?"** (Confidence? Warmth? Intelligence? Style? Calm authority?)

### Contrast Spectrum
56. **"On a scale, where does your brand sit?"** Present as pairs:
    - Loud ←→ Quiet
    - Warm ←→ Cool
    - Organic ←→ Engineered
    - Accessible ←→ Exclusive
    - Playful ←→ Serious
    - Traditional ←→ Disruptive
    - Complex ←→ Simple
    - Fast ←→ Deliberate

### Narrative
57. **"What is the one thing you want someone to feel in the first 2 seconds of seeing this logo?"** Force a single answer, not a list.
58. **"If a stranger saw this logo with no context, what single word should come to mind?"**
59. **"What would make you look at the final logo and say 'that is exactly right'? What specific quality would trigger that reaction?"**
60. **"What would make you look at it and say 'that is completely wrong'? What would be a dealbreaker?"**

### Competitive Positioning
61. **"Your competitor's logo says [X]. What should yours say differently?"**
62. **"If all the logos in your industry were lined up, how should yours stand out? Through color? Shape? Simplicity? Boldness?"**

### Future-Proofing
63. **"Where do you see this brand in 5 years? Will it expand into new products, markets, or audiences?"**
64. **"Is there a chance the brand name or offering will evolve? Should the logo be adaptable?"**

### Decision-Making Style
65. **"When choosing between two logo options, what will you evaluate first: how it looks or how it makes you feel?"** (This tells you whether to lead with aesthetics or rationale in your presentation.)
66. **"Do you prefer to see many options and narrow down, or fewer polished options to choose between?"**

## Interview Strategy & Adaptive Rules

### Round Structure

**Round 1 (5 questions max):** Tier 1 essentials. Get the foundation.
**Round 2 (4-6 questions):** Tier 2 + relevant Tier 3 + 1-2 Tier 6 emotional probes. Deepen understanding.
**Round 3 (2-4 questions):** Remaining Tier 3 + Tier 4 situational + Tier 5 industry. Fill gaps.

### When to Use Tier 6 (Emotional Probing)

- When the user's Tier 1 emotional keywords feel generic ("modern, clean, professional" -- 80% of brands say this)
- When the user struggles to articulate their vision
- When competing priorities emerge ("we want bold AND elegant")
- When differentiating from competitors with similar positioning
- Always include question 57 or 58 in Round 2 -- it anchors the entire design direction

### Adaptive Rules

- If the user provides a detailed brief upfront, skip questions already answered
- If the user says "surprise me," still ask Tier 1 questions 1-4 plus one Tier 6 emotional probe
- If the user is impatient, compress to: name, industry, 3 mood words, logo type, color preference. Minimum viable brief.
- If the user provides inspiration images, analyze them before asking style questions -- extract style, color, type, and weight observations
- If answers contradict (e.g., "minimalist" + "lots of detail"), surface the tension: "I notice a pull between minimalism and richness -- which should win when they conflict?"

### Question Delivery Style

- Present options when possible (don't make the user generate answers from nothing)
- Use concrete examples (reference logos the user likely knows)
- Group related questions (don't jump between topics)
- Acknowledge and build on previous answers: "Since you mentioned wanting something futuristic, let me ask..."
- Offer professional perspective: "Given your industry, I'd suggest considering... what do you think?"
- Keep questions conversational, not interrogative
- If a question reveals strong feelings, dig in: "Tell me more about why that matters to you"

## Brief Template

After completing discovery, synthesize into this format and present for confirmation:

```
## Design Brief: [Brand Name]

**Brand**: [Full name + tagline]
**Industry**: [Sector / description]
**Logo type**: [Selected type or exploring multiple]
**Mood**: [3-5 emotional keywords, refined from interview]
**Core feeling**: [The single-word answer from question 57/58]
**Visual style**: [Selected style(s)]
**Color direction**: [Specified colors with hex OR mood-based direction]
**Audience**: [Target description with psychographic detail]
**Key metaphors/symbols**: [What to explore visually]
**Must avoid**: [Explicit exclusions]
**Inspiration**: [Referenced logos + what user liked + what to extract from each]
**Brand spectrum**: [Key positions on the contrast spectrums from Q56]
**Primary use**: [Main application context]
**Size requirements**: [Small-size needs]
**Layouts**: [Which formats needed]
**Concepts**: [Number requested]
**Decision style**: [Aesthetic-first or rationale-first, many options vs few polished]
**Special notes**: [Rebrand context, multi-brand, international, future plans]
```

Always present the brief for user confirmation before beginning any design work.
