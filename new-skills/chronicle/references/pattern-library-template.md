# Pattern Library Template

Longitudinal tracker for recurring behavioral, emotional, and cognitive patterns in a user's journal. Patterns often get named by the user (e.g., "Plan Sabotage"); capture them here and reference on future entries.

---

## How to Use

- **Chronicle (AI):** Reference when similar patterns appear. Note when improving or worsening. Add new patterns as the user names or demonstrates them.
- **User:** Review periodically for self-awareness. Share with therapist if relevant.
- **Therapist (if shared):** Provides context across time in the user's own language; tracks what interventions help.
